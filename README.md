# Todo List
Simple TodoList for adding, removing items from a list. Created with [React](https://reactjs.org/)

## Requirements

- [npm](https://www.npmjs.com/)
- [React](https://reactjs.org/)

## How to run
In the project directory, type command:

```bash
npm start
```
